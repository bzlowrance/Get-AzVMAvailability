# Traffic Data

Automatically collected by the [collect-traffic workflow](../.github/workflows/collect-traffic.yml).
Run daily via GitHub Actions cron.

## Files
- `views.csv` — Daily page views
- `clones.csv` — Daily git clones
- `stars.csv` — Star history with timestamps
- `referrers.csv` — Referral sources (snapshot per day)
- `paths.csv` — Popular content (snapshot per day)
- `repo-stats.csv` — Daily repo metrics (stars, forks, watchers)
